# Лабораторная работа 2:  STL 

STL предоставляет готовые контейнеры (vector, map и т.д.), алгоритмы (sort, find) и итераторы для эффективной работы с данными. В C эквиваленты отсутствуют, требуя ручной реализации. STL использует шаблоны для обобщенности. 

 

Задания 

N=3 параметра: (1) контейнер (vector/list/map), (2) размер (5-15), (3) алгоритм (sort/find/transform). Хеш: первые hex для контейнера (0-5:vector, 6-A:list, B-F:map); следующие для размера (mod 11 +5); остальные для алгоритма. 

Пример для "Иванов Иван Иванович": контейнер=list, размер=10, алгоритм=sort. 

 

1: Универсальная работа с алгоритмом 
Создайте [контейнер] размером [размер] и заполните его числами от 1 до [размер]. В зависимости от заданного алгоритма: 
Если алгоритм sort — отсортируйте контейнер в порядке убывания. 
Если алгоритм find — найдите число 5 и выведите его индекс (или сообщение, что не найдено). 
Если алгоритм transform — замените каждый элемент его квадратом. 

2: Работа с пользовательским вводом 
Напишите функцию, которая: 
Запрашивает у пользователя [размер] чисел и заполняет ими [контейнер]. 
Применяет заданный [алгоритм]: 
Для sort — сортирует по возрастанию и выводит минимальный и максимальный элементы. 
Для find — запрашивает у пользователя значение для поиска и выводит, сколько раз оно встречается (для map — ищет по ключу). 
Для transform — заменяет все отрицательные числа на их модуль и выводит сумму всех элементов. 
Обрабатывает случай, если контейнер пуст или алгоритм не может быть применен. 

3: Компараторы и предикаты 
Реализуйте программу, которая: 
Заполняет [контейнер] [размер] случайными строками вида "item_X", где X — рандомное число от 1 до 100. 
В зависимости от алгоритма: 
Для sort — создайте компаратор, который сортирует строки по длине (от коротких к длинным), а при равной длине — лексикографически. 
Для find — используйте предикат для поиска первой строки, содержащей цифру '5'. 
Для transform — преобразуйте все строки к верхнему регистру и сохраните результат в новый вектор. 
Выведите исходный и результирующий контейнеры. 

4: Многоуровневая сортировка с пользовательскими типами 
Создайте структуру Student с полями: name (string), group (int), grade (double). Заполните [контейнер] [размер] объектами Student со случайными данными. 
В зависимости от алгоритма: 
Если sort — реализуйте сортировку по трем уровням: сначала по группе, затем по убыванию оценки, затем по имени. Используйте комбинацию компараторов. 
Если find — найдите всех студентов с оценкой выше 4.5, сгруппируйте их по группам и выведите статистику (количество, средняя оценка по группе). 
Если transform — создайте из контейнера студентов map<int, vector<string>>, где ключ — номер группы, а значение — список имен студентов этой группы. 

5: Конвейерная обработка с итераторами и вспомогательными алгоритмами 
Реализуйте многоэтапную обработку данных: 
Создайте [контейнер] и заполните его [размер] случайными числами от -50 до 50. 
Этап 1 (подготовка): 
Для sort — удалите все дубликаты (std::unique) после сортировки. 
Для find — замените все отрицательные числа на 0 (std::replace_if). 
Для transform — оставьте только числа, кратные 3 (std::copy_if в новый контейнер). 
Этап 2 (основной алгоритм): 
Для sort — к уникальным числам примените сортировку по модулю числа. 
Для find — в модифицированном контейнере найдите первое число, попадающее в интервал [10, 20]. 
Для transform — к отфильтрованным числам примените возведение в квадрат. 
Этап 3 (анализ): Вычислите и выведите: 
Для sort — разницу между максимальным и минимальным элементом. 
Для find — индекс найденного элемента (или -1, если не найден). 
Для transform — произведение всех элементов. 
Используйте итераторы на всех этапах, избегайте индексации там, где это возможно.
