# Лабораторная работа 1: Шаблоны 
Шаблоны в C++ позволяют создавать обобщенный код, который работает с разными типами данных без дублирования. Это ключевая особенность, отсутствующая в C, где для похожих функций нужны макросы или дубликатный код. Шаблоны компилируются в конкретные экземпляры для каждого типа, обеспечивая type-safety и производительность.

Пример:
template <typename T>
T max(T a, T b) {
    return (a > b) ? a : b;
}

int main() {
    int x = max(3, 7);      // Использует int
    double y = max(2.5, 4.1); // Использует double
    return 0;
}

Здесь функция max работает для любых сравнимых типов.
