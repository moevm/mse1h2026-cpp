# Шифр Цезаря
1. Общее описание
Разработать программу на языке C++, реализующую шифр Цезаря — один из простейших методов шифрования, при котором каждый символ в тексте заменяется символом, находящимся на некоторое фиксированное число позиций левее или правее от него в алфавите.

2. Функциональные требования
Программа должна:

2.1. Запросить у пользователя:

Исходный текст для шифрования (может содержать буквы русского или английского алфавита, пробелы и знаки препинания)

Ключ сдвига (целое число, может быть положительным или отрицательным)

2.2. Выполнить шифрование текста в соответствии со следующими правилами:

Заглавные буквы должны оставаться заглавными после сдвига

Строчные буквы должны оставаться строчными после сдвига

Сдвиг должен быть циклическим (после 'Я' идет 'А' для русского алфавита, после 'Z' идет 'A' для английского)

Пробелы, знаки препинания и прочие символы, не являющиеся буквами, должны оставаться без изменений

Русские и английские буквы обрабатываются независимо (сдвиг в пределах своего алфавита)

2.3. Вывести на экран:

Исходный текст

Использованный ключ сдвига

Результат шифрования (зашифрованный текст)

3. Требования к реализации
3.1. Программа должна быть реализована в одном файле caesar_cipher.cpp.

3.2. Обязательно использование функций для организации кода:

Функция char shiftChar(char ch, int shift, const std::string& alphabet) — сдвигает один символ в пределах заданного алфавита

Функция std::string encrypt(const std::string& text, int shift) — обрабатывает всю строку

3.3. Поддержка двух алфавитов:

Русский: АБВГДЕЁЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ (33 буквы, включая Ё)

Английский: ABCDEFGHIJKLMNOPQRSTUVWXYZ (26 букв)

3.4. Корректная обработка граничных случаев:

Отрицательный сдвиг (сдвиг влево)

Сдвиг, превышающий длину алфавита (например, сдвиг 35 для английского алфавита эквивалентен сдвигу 9, так как 35 mod 26 = 9)

Пустая строка на входе
