## Работа с матрицами

Вам предстоит протестировать модуль, реализующий операции с матрицами. Модуль содержит функции для сложения, умножения, 
транспонирования и вычисления детерминанта матриц. Все матрицы представлены в виде двумерных массивов (или структур, содержащих размеры и данные). 
Вы должны провести тестирование "серого ящика" (gray box), что означает, что вам известна внутренняя структура: модуль состоит из отдельных функций для каждой операции, 
которые могут вызывать друг друга (например, умножение может использовать вспомогательные функции). Ваша задача — разработать набор тестов, 
проверяющих корректность работы модуля на двух уровнях (функции по отдельности, функции в пайплайне). Не забывайте о том, что важно тестировать граничные случаи.


Вам предоставлены следующие функции (реализация скрыта, но известна спецификация):

1.  `Matrix* add_matrix(Matrix* A, Matrix* B)` — главная функция сложения матриц. Возвращает новую матрицу — сумму A и B. Возвращает NULL при ошибке (несовпадение размеров).
2.  `Matrix* multiply_matrix(Matrix* A, Matrix* B)` — главная функция умножения матриц. Возвращает новую матрицу — произведение A и B. Возвращает NULL при ошибке (несогласованные размеры)
3.  `Matrix* transpose_matrix(Matrix* A)` — функция транспонирования. Возвращает новую матрицу — транспонированную A
4.  `double determinant_matrix(Matrix* A)` — функция вычисления детерминанта для квадратной матрицы. Возвращает значение детерминанта. Возвращает NULL при ошибке


#### 1. Модульное тестирование (Unit Tests)

Протестируйте изолированно каждую функцию модуля. Так как это "серый ящик" (gray box), Вы можете вызывать их напрямую.

*   Тестирование сложения (`add_matrix`)
*   Тестирование умножения (`multiply_matrix`)
*   Тестирование транспонирования (`transpose_matrix`)
*   Тестирование детерминанта (`determinant_matrix`)


#### 2. Интеграционное и сквозное тестирование 

Проверьте корректность взаимодействия функций модуля при выполнении цепочек операций.

*   Создайте тесты, проверяющие свойство ассоциативности сложения: `(A + B) + C == A + (B + C)`
*   Создайте тесты, проверяющие дистрибутивность умножения: `A * (B + C) == (A * B) + (A * C)`
*   Создайте тесты, проверяющие свойство транспонирования произведения: `(A * B)^T == B^T * A^T`
*   Создайте тесты, проверяющие, что детерминант произведения равен произведению детерминантов: `det(A * B) == det(A) * det(B)` (для квадратных матриц)
*   Придумайте свои тесты
