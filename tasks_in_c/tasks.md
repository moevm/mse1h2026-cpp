# Задачи по тестированию на языке C

## Задача 1
**Дана функция:** `int is_even(int n)` (должна возвращать 1, если число четное, 0 в противном случае).  
**Задание:** Напишите тест, который проверяет работу функции на нескольких числах (минимум 4: четное положительное, нечетное положительное, ноль, отрицательное четное).

## Задача 2
**Дана функция:** `int max2(int a, int b)` (должна возвращать большее из двух чисел).  
**Задание:** Напишите тесты, проверяющие случаи: `a > b`, `a < b`, `a == b`.

## Задача 3
**Дана функция:** `int my_abs(int x)` (должна возвращать модуль числа).  
**Задание:** Протестируйте функцию на положительном, отрицательном числе и нуле. Отдельно проверьте граничное значение `INT_MIN` (если ваша система его поддерживает, подумайте, как его корректно сравнить).

## Задача 4
**Дана функция:** `int sum_to_n(int n)` (должна возвращать сумму 1 + 2 + ... + n).  
**Задание:** Напишите тесты для `n = 0`, `n = 1`, `n = 5` (ожидаемая сумма 15) и отрицательного `n` (что должна возвращать функция? Скорее всего, 0).

## Задача 5
**Дана функция:** `int sign(int x)` (должна возвращать -1 для отрицательных, 0 для нуля, 1 для положительных).  
**Задание:** Напишите тесты, проверяющие все три случая.

## Задача 6
**Дана функция:** `int is_leap(int year)` (должна возвращать 1, если год високосный).  
**Задание:** Напишите тесты для проверки разными годами.

## Задача 7
**Дана функция:** `int days_in_month(int month, int year)` (должна возвращать количество дней в месяце с учетом года).  
**Задание:** Протестируйте функцию для разных месяцев разных лет (не забудьте, что в феврале может быть различное количество дней в зависимости от года).

## Задача 8   (ДОПИСАТЬ)
**Дана функция:** `int solve_quadratic(double a, double b, double c, double* x1, double* x2)` (возвращает количество корней или -1, если `a == 0`).  
**Задание:** Напишите тесты для случаев: два корня (`x^2 - 3x + 2 = 0`), один корень (`x^2 - 2x + 1 = 0`), нет корней (`x^2 + 1 = 0`), линейное уравнение (`a = 0`).

## Задача 9
**Дана функция:** `int triangle_type(int a, int b, int c)` (по длинам сторон a, b, c должна определять треугольник: 0 - не треугольник, 1 - остроугольный, 2 - прямоугольный, 3 - тупоугольный).  
**Задание:** Проверьте различные комбинации длин сторон, подумайте над крайними случаями.

## Задача 10
**Дана функция:** `double calculate_discount(double price, int percent)` (должна возвращать цену с учетом скидки).  
**Задание:** Протестируйте вычисление цены со скидкой. Из-за погрешностей вычислений чисел с плавающей точкой, сравнение должно быть с допуском (epsilon), подберите его самостоятельно.

## Задача 11
**Дана функция:** `long long factorial(int n)` (должна возвращать факториал заданного числа).  
**Задание:** Напишите тесты для различных входных данных, подумайте над крайними случаями.  

## Задача 12
**Дана функция:** `int fib(int n)` (должна возвращать число Фиббоначи на заданном индексе).  
**Задание:** Напишите тесты для различных входных данных, подумайте над крайними случаями. 

## Задача 13
**Дана функция:** `int count_digits(int n)` (должна возвращать количество цифр в десятичной записи числа).  
**Задание:** Протестируйте на различных числах. 0 (1 цифра), 5 (1 цифра), 123 (3 цифры), -456 (3 цифры, знак не считается).

## Задача 14
**Дана функция:** `int sum_digits(int n)` (должна возвращать сумму цифр заданного числа в десятичной записи).  
**Задание:** Протестируйте на различных числах. 0 -> 0, 123 -> 6, 999 -> 27, -123 -> 6 (по модулю).

## Задача 15
**Дана функция:** `int is_prime(int n)` (должна возвращать 1, если число простое, иначе 0).  
**Задание:** Протестируйте на различных числах, подумайте над граничными случаями. 0, 1 (не простые), 2 (простое), 3 (простое), 4 (составное), 17 (простое), 25 (составное).

## Задача 16
**Дана функция:** `int array_max(int* arr, int size)` (должна возвращать максимальное число заданном массиве).  
**Задание:** Протестируйте данную функцию. Подумайте над крайними случаями (например, массив из одного элемента, массив с отрицательными числами и тд).

## Задача 17
**Дана функция:** `void reverse_array(int* arr, int size)` (должна разворачивать массив в обратную сторону).  
**Задание:** Протестируйте на различных массивах (создайте массив, вызовите функцию, проверьте, что порядок элементов изменился на противоположный), подумайте над крайними случаями.

## Задача 18
**Дана функция:** `int find_index(int* arr, int size, int value)` (должна возвращать индекс первого вхождения числа или -1).  
**Задание:** Протестируйте на различных массивах и значениях (элемент есть в начале, в середине, в конце; элемента нет; пустой массив (size = 0) и тд).

## Задача 19
**Дана функция:** `int count_even(int* arr, int size)` (возвращает количество четных чисел в массиве).  
**Задание:** Протестируйте на различных массивах. Примеры тесты: все четные, все нечетные, смешанный массив, пустой массив (ожидаем 0).

## Задача 20
**Дана функция:** `void bubble_sort(int* arr, int size)` (функция реализует сортировку пузырьком).  
**Задание:** Протестируйте корректность сортировки на различных массивах. Подумайте над крайними случаями. Факт, что функция реализует именно "пузырек" проверить не получится, смотрите на функцию как на абстрактую сортировку в черном ящике.
