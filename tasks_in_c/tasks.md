# Задачи по тестированию на языке C

## Задача 1
**Дана функция:** `int is_even(int n)` (по задумке функция должна возвращать 1, если число четное, 0 в противном случае).  
**Задание:** Напишите тест, который проверяет работу функции на нескольких числах (четное положительное, нечетное положительное, ноль, отрицательное четное и т.д.).

## Задача 2
**Дана функция:** `int max2(int a, int b)` (должна возвращать большее из двух чисел).  
**Задание:** Напишите тесты, проверяющие все случаи: `a > b`, `a < b`, `a == b`.

## Задача 3
**Дана функция:** `int my_abs(int x)` (должна возвращать модуль числа).  
**Задание:** Протестируйте функцию на положительном, отрицательном числе и нуле. Отдельно проверьте граничное значение `INT_MIN` (если ваша система его поддерживает, подумайте, как его корректно сравнить).

## Задача 4
**Дана функция:** `int sum_to_n(int n)` (должна возвращать сумму 1 + 2 + ... + n).  
**Задание:** Напишите тесты для различных `n`, в том числе для отрицательных (что должна возвращать функция? Скорее всего, 0).

## Задача 5
**Дана функция:** `int sign(int x)` (проверка знака: должна возвращать -1 для отрицательных, 0 для нуля, 1 для положительных).  
**Задание:** Напишите тесты, проверяющие все три случая: отрицательные, ноль, положительные.

## Задача 6
**Дана функция:** `int is_leap(int year)` (должна возвращать 1, если год високосный).  
**Задание:** Напишите тесты для проверки разными годами (делящийся на 400, делящийся на 100 но не на 400, делящийся на 4, но не на 100, невисокосный).

## Задача 7
**Дана функция:** `int days_in_month(int month, int year)` (должна возвращать количество дней в месяце с учетом года).  
**Задание:** Протестируйте функцию для разных месяцев разных лет (не забудьте, что в феврале может быть различное количество дней в зависимости от года + проверьте некорректные номера месяцев).

## Задача 8
**Дана функция:** `int triangle_type(int a, int b, int c)` (по длинам сторон a, b, c должна определять треугольник: 0 - не треугольник, 1 - остроугольный, 2 - прямоугольный, 3 - тупоугольный).  
**Задание:** Проверьте различные комбинации длин сторон, подумайте над крайними случаями (равносторонний, равнобедренный, вырожденный треугольник со сторонами 1, 1, 2).

## Задача 9
**Дана функция:** `double calculate_discount(double price, int percent)` (должна возвращать цену с учетом скидки).  
**Задание:** Протестируйте вычисление цены со скидкой. Из-за погрешностей вычислений чисел с плавающей точкой, сравнение должно быть с допуском (epsilon), подберите его самостоятельно. Проверьте граничные значения процентов (0% и 100%).

## Задача 10
**Дана функция:** `long long factorial(int n)` (должна возвращать факториал заданного числа).  
**Задание:** Напишите тесты для различных входных данных (`0! = 1`, `1! = 1`, `5! = 120`). Подумайте над крайними случаями: отрицательное число (что будет?) и очень большое `n`, при котором произойдет переполнение (тест на попытку переполнения).

## Задача 11
**Дана функция:** `int fib(int n)` (должна возвращать число Фибоначчи на заданном индексе, где `F(0) = 0, F(1) = 1`).  
**Задание:** Напишите тесты для различных входных данных (`n = 0`, `n = 1`, `n = 10` -> 55). Подумайте над крайними случаями: отрицательное `n` и попытка переполнения `int`.

## Задача 12
**Дана функция:** `int count_digits(int n)` (должна возвращать количество цифр в десятичной записи числа, игнорируя знак).  
**Задание:** Протестируйте на различных числах: `0` (1 цифра), `5` (1 цифра), `123` (3 цифры), `-456` (3 цифры, знак не считается). Проверьте максимальное и минимальное значение `int`.

## Задача 13
**Дана функция:** `int sum_digits(int n)` (должна возвращать сумму цифр заданного числа в десятичной записи).  
**Задание:** Протестируйте на различных числах: `0` -> 0, `123` -> 6, `999` -> 27, `-123` -> 6.

## Задача 14
**Дана функция:** `int is_prime(int n)` (должна возвращать 1, если число простое, иначе 0).  
**Задание:** Протестируйте на различных числах, подумайте над граничными случаями. Проверьте отрицательные числа.

## Задача 15
**Дана функция:** `int array_max(int* arr, int size)` (должна возвращать максимальное число в заданном массиве).  
**Задание:** Протестируйте данную функцию. Подумайте над крайними случаями (массив из одного элемента, массив с отрицательными числами, массив, где максимум находится в начале/середине/конце). Что должно происходить при `size == 0`?

## Задача 16
**Дана функция:** `void reverse_array(int* arr, int size)` (должна разворачивать массив в обратную сторону).  
**Задание:** Протестируйте на различных массивах (создайте массив, вызовите функцию, проверьте, что порядок элементов изменился на противоположный). Подумайте над крайними случаями: массив из одного элемента, массив из четного/нечетного количества элементов, пустой массив (`size = 0`).

## Задача 17
**Дана функция:** `void swap_ranges(int* arr1, int* arr2, int start, int end)` (функция должна обменять местами элементы двух массивов `arr1` и `arr2` с индексами `start` до `end` включительно).  
**Задание:** Напишите тесты, проверяющие корректность обмена в следующих случаях: обмен части массивов одинаковой длины, обмен с выходом за границы (если `end` больше размера массива - функция должна ничего не делать или обрабатывать это как ошибку), обмен, когда `start > end` (функция не должна ничего менять), обмен на пустом диапазоне.

## Задача 18
**Дана функция:** `int find_index(int* arr, int size, int value)` (должна возвращать индекс первого вхождения числа или -1, если элемент не найден).  
**Задание:** Протестируйте на различных массивах и значениях (элемент есть в начале, в середине, в конце; элемента нет; пустой массив (`size = 0`) и т.д.).

## Задача 19
**Дана функция:** `int count_even(int* arr, int size)` (возвращает количество четных чисел в массиве).  
**Задание:** Протестируйте на различных массивах: все четные, все нечетные, смешанный массив, пустой массив (ожидаем 0). Проверьте работу с отрицательными четными числами.

## Задача 20
**Дана функция:** `int string_length(const char* str)` (должна возвращать длину строки, не считая завершающий нуль-символ).  
**Задание:** Протестируйте функцию на различных строках: пустая строка, строка из одного символа, строка из нескольких символов. Что произойдет, если передать `NULL`? (Это зависит от реализации, тест должен это учесть).

## Задача 21
**Дана функция:** `void string_copy(char* dest, const char* src)` (копирует строку из `src` в `dest`, предполагая, что `dest` достаточно велик).  
**Задание:** Напишите тест, который проверяет, что после копирования `dest` содержит точно такую же строку, как `src`. Проверьте на пустой строке.

## Задача 22
**Дана функция:** `void string_concatenate(char* dest, const char* src)` (должна приписывать строку `src` в конец строки `dest`).  
**Задание:** Протестируйте конкатенацию двух непустых строк, пустой строки к непустой, непустой к пустой, двух пустых строк.

## Задача 23
**Дана функция:** `int string_compare(const char* str1, const char* str2)` (должна возвращать 0, если строки равны, отрицательное, если `str1 < str2`, положительное в противном случае).  
**Задание:** Напишите тесты для сравнения равных строк, разных строк (лексикографически), строк разной длины.

## Задача 24
**Дана функция:** `char* find_substring(const char* str, const char* substr)` (должна возвращать указатель на первое вхождение `substr` в `str` или `NULL`, если подстроки нет).  
**Задание:** Протестируйте нахождение подстроки в начале, середине и конце строки. Проверьте случай, когда подстрока длиннее строки и когда подстрока пустая.

## Задача 25
**Дана функция:** `int* create_array(int size, int value)` (должна выделять память под массив из `size` элементов и заполнять его `value`).  
**Задание:** Напишите тесты, проверяющие, что массив создался корректно (все элементы равны `value`). Проверьте граничные случаи: `size = 0`, `size = 1`. Что происходит при ошибке выделения памяти (возврат `NULL`)?

## Задача 26
**Дана функция:** `int popcount(int n)` (должна возвращать количество единиц в двоичном представлении числа).  
**Задание:** Протестируйте на различных числах: `0` -> 0, `1` -> 1, `3` (0b11) -> 2, `255` (0b11111111) -> 8.

## Задача 27
**Дана функция:** `double average(double* arr, int size)` (должна возвращать среднее арифметическое элементов массива).  
**Задание:** Проверьте на обычных массивах. Проверьте массив из одного элемента. Проверьте массив из одинаковых чисел. Что происходит при `size = 0`?

## Задача 28
**Дана функция:** `int is_palindrome(const char* str)` (должна возвращать 1, если строка является палиндромом, 0 в противном случае).  
**Задание:** Напишите тесты для пустой строки (обычно палиндром), строки из одного символа, строк с четным и нечетным количеством символов, строки, не являющейся палиндромом.

## Задача 29
**Дана структура:** `typedef struct { int x; int y; } Point;` и функция `int is_point_in_rectangle(Point p, Point top_left, Point bottom_right)` (определяет, находится ли точка внутри прямоугольника: -1 не находится, 0 на границе, 1 находится внутри).  
**Задание:** Проверьте точку внутри, снаружи, на границе прямоугольника. Подумайте о корректности границ (включительно или нет).

## Задача 30
**Дана функция:** `void swap_pointers(int** a, int** b)` (должна менять местами значения двух указателей).  
**Задание:** Напишите тест, который создает два указателя на разные переменные, вызывает функцию и проверяет, что теперь они указывают на переменные друг друга.

## Задача 31
**Дана функция:** `int is_sorted(int* arr, int size)` (должна возвращать 1, если массив отсортирован по возрастанию).  
**Задание:** Протестируйте на уже отсортированном массиве, несортированном массиве, массиве из одного элемента, пустом массиве. 

## Задача 32
**Дана функция:** `int* merge_arrays(int* src1, int size1, int* src2, int size2)` (должна сливать два отсортированных массива в один отсортированный `dest`).  
**Задание:** Протестируйте слияние массивов разной длины, включая пустые массивы. Проверьте, что результат действительно отсортирован и содержит все элементы.

## Задача 33
**Дана функция:** `long long power(int base, int exp)` (должна возводить `base` в степень `exp`).  
**Задание:** Проверьте возведение в степень 0 (должно быть 1), в степень 1, в положительную степень (`2^10 = 1024`). Проверьте обработку отрицательных степеней (функция с целочисленным возвратом обычно не должна их принимать, либо возвращать 0) и переполнение.

## Задача 34
**Дана функция:** `char* to_uppercase(const char* str)` (должна возвращать новую строку в верхнем регистре).  
**Задание:** Проверьте, что функция корректно преобразует строчные буквы, не меняет заглавные и не буквенные символы. Проверьте на пустой строке. Не забудьте проверить, что исходная строка не изменилась.

## Задача 35

**Задание:** Протестируйте на пустой строке, строке с одним словом, строке с несколькими словами, строке с лидирующими и завершающими пробелами, строке с несколькими пробелами подряд.

## Задача 36

## Задача 37

## Задача 38

## Задача 39

## Задача 40
