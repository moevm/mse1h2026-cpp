# Задание

Напишите шаблонную функцию `add`, которая складывает два числа. Используйте стандартный концепт, чтобы ограничить принимаемые типы только **целыми числами** (int, long, short и т.д.).

**1) Подключение**
- Подключите заголовок `<concepts>`.

**2) Функция `add(a, b)`**
- Принимает два аргумента одного типа `T`.
- Ограничение: `T` должен удовлетворять концепту `std::integral`.
- Возвращает сумму `a + b`.

**3) Проверка**
- Код `add(5, 10)` должен компилироваться.
- Код `add(5.5, 2.3)` или `add("a", "b")` **не должен** компилироваться.
