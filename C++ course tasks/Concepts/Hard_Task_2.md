# Задание

Напишите **свой собственный концепт** `Hashable`, который проверяет, можно ли вычислить хеш от объекта с помощью стандартной функции `std::hash`.

**1) Концепт `Hashable<T>`**
- Должен проверять валидность выражения `std::hash<T>{}(x)`, где `x` — объект типа `T`.
- Должен проверять, что результат этого выражения конвертируется в `std::size_t`.

**2) Функция `printHash(const T& val)`**
- Принимает любой тип, удовлетворяющий `Hashable`.
- Выводит хеш значения в консоль.
