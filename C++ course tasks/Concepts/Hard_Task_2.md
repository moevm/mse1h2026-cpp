# Задание 

Напишите концепт `MatrixLike<T>`, который проверяет, что тип ведет себя как двумерный массив (матрица).

**1) Требования концепта**
- У типа `T` должны быть методы `.rows()` и `.cols()`, возвращающие целое число.
- Должен поддерживаться оператор доступа по индексу `m[i]`, который возвращает строку.
- Строка, в свою очередь, должна поддерживать `row[j]`, возвращая значение элемента.

**2) Функция `printDiagonal(const M& matrix)`**
- Работает только с `MatrixLike`.
- Выводит элементы `m[i][i]`.
