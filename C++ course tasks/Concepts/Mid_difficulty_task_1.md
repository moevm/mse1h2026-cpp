# Задание

Реализуйте функцию `safe_clamp`, которая ограничивает значение диапазоном `[min, max]`. Используйте стандартные концепты для гарантии корректности операций сравнения.

**1) Концепт `std::totally_ordered`**
- Гарантирует, что для типа определены операторы `<`, `>`, `<=`, `>=`.

**2) Функция `safe_clamp(val, min, max)`**
- Все три аргумента должны быть одного типа `T`.
- Ограничение: `T` должен быть `std::totally_ordered`.
- Возвращает `val`, если оно внутри диапазона.
- Возвращает `min`, если `val < min`.
- Возвращает `max`, если `val > max`.
