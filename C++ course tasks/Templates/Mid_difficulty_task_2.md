
# Задание

Вам нужно реализовать шаблонный класс `Cache`, который представляет простой кэш фиксированного размера для хранения пар ключ-значение. Класс должен работать с типами `int`, `std::string` в качестве ключей и значений.

**Класс должен включать:**
- `typename K` - тип ключа (может быть `int`, `std::string`)
- `typename V` - тип значения (может быть `int`, `std::string`)
- `size_t MaxSize` - максимальный размер кэша (фиксированный, задается при компиляции)

**Методы:**
**1) Конструктор**
- `Cache()` - конструктор по умолчанию, создает пустой кэш

**2) Основные операции с кэшем**
- `void put(const K& key, const V& value)` - добавляет пару ключ-значение в кэш
    - Если ключ уже существует, обновляет значение
    - Если кэш полон и ключ новый, выбрасывает исключение `std::overflow_error` с сообщением "Cache is full"
- `V get(const K& key) const` - возвращает значение по ключу
    - Если ключ не найден, выбрасывает исключение `std::out_of_range` с сообщением "Key not found in cache"
- `bool contains(const K& key) const` - проверяет, существует ли ключ в кэше
    - Возвращает `true`, если ключ найден, иначе `false`

**3) Удаление элементов**
- `void remove(const K& key)` - удаляет элемент по ключу
    - Если ключ не найден, ничего не делает
- `void clear()` - полностью очищает кэш

**4) Вспомогательные методы**
- `size_t size() const` - возвращает текущее количество элементов в кэше
- `size_t capacity() const`- возвращает максимальный размер кэша (MaxSize)
- `bool empty() const` - проверяет, пуст ли кэш
- `std::vector<K> keys() const` - возвращает список всех ключей, хранящихся в кэше
