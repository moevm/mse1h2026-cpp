
# Задание:

Нужно реализовать шаблонный класс `Calculator`, который выполняет базовые арифметические операции над двумя значениями. Класс должен иметь общую реализацию для числовых типов, но специализированное поведение для целых чисел (`int`) и строк
(`std::string`).

**Класс должен включать:**
- `typename T` - тип операндов

**Методы:**

**1) Основные операции**
- `T add(T a, T b) const` - сложение двух значений
- `T subtract(T a, T b) const` - вычитание (a - b)
- `T multiply(T a, T b) const` - умножение
- `T divide(T a, T b) const` - деление (a / b)
    - Если b равно 0, выбрасывает исключение `std::runtime_error` с сообщением "Division by zero"

**2) Дополнительные методы**
- `T max(T a, T b) const` - возвращает большее из двух значений
- `T min(T a, T b) const` - возвращает меньшее из двух значений


**Специализация 1:**
Для целых чисел добавить и изменить:
- **Новый метод:** `int mod(int a, int b) const` - возвращает остаток от деления a на b
    - Если b равно 0, выбрасывает исключение `std::runtime_error` с сообщением "Modulo by zero"
- `divide()` - выполняет целочисленное деление (отбрасывает дробную часть)

**Специализация 2:**
Для строк изменить поведение следующих методов:
- `add()` - выполняет конкатенацию строк (a + b)
- `subtract()` - удаляет первое вхождение подстроки b из строки a
    - Если b не найдена, возвращает исходную строку без изменений
- `multiply()` - повторяет строку a b раз (b рассматривается как целое число)
    - Если b ≤ 0, возвращает пустую строку
- `divide()` - не поддерживается, всегда выбрасывает исключение `std::logic_error` с сообщением "String division is not supported"
- `max()` - возвращает лексикографически большую строку
- `min()` - возвращает лексикографически меньшую строку
