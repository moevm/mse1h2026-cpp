
# Задание

Вам нужно реализовать шаблонный класс `Stack`, который представляет стек фиксированного размера. Класс должен работать с типами `int`, `double` и `std::string` и позволять задавать размер стека через параметр шаблона.

**Класс должен включать:**
- `typename T` - тип элементов, хранящихся в стеке (может быть `int`, `double` или `std::string`)
- `size_t N` - максимальный размер стека (фиксированный, задается при компиляции)

**Методы:**
**1) Конструктор**
- `Stack()` - конструктор по умолчанию, создает пустой стек

**2) Основные операции стека**
- `void push(const T& value)` - добавляет элемент на вершину стека
    - Если стек полон, ничего не делает (или можно выбросить исключение — по желанию)
- `T pop()` - удаляет и возвращает элемент с вершины стека
    - Если стек пуст, выбрасывает исключение `std::runtime_error` с сообщением "Stack is empty"
- `T top() const` - возвращает элемент с вершины без удаления
    - Если стек пуст, выбрасывает исключение `std::runtime_error`

**3) Вспомогательные методы**
- `bool empty() const` - проверяет, пуст ли стек
- `size_t size() const` - возвращает текущее количество элементов в стеке
- `size_t capacity() const` - возвращает максимальный размер стека (N)
