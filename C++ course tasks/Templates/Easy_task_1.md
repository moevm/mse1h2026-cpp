
# Задание:

Вам нужно реализовать четыре шаблонные функции для выполнения базовых арифметических операций. Функции должны работать с типами `int` и `double`.

**1) Функция `max`**
- Принимает два значения одного типа
- Возвращает большее из них
- Если значения равны, возвращает первое

**2) Функция `min`**
- Принимает два значения одного типа
- Возвращает меньшее из них
- Если значения равны, возвращает первое

**3) Функция `abs`**
- Принимает одно значение
- Возвращает его абсолютное значение (модуль числа)
- Для отрицательных чисел возвращает положительное, для положительных - само число

**4) Функция `clamp`**
- Принимает три значения одного типа: `value`, `low`, `high`
- Возвращает:
    - `low`, если `value` меньше `low`
    - `high`, если `value` больше `high`
    - `value` в остальных случаях
