
# Задание

Реализуйте класс `Fraction`, представляющий обыкновенную дробь (числитель и знаменатель). Класс должен всегда хранить дробь в несократимом виде (например, 2/4 должно храниться как 1/2).

**1) Приватные поля**
- `int numerator` (числитель)
- `int denominator` (знаменатель)

**2) Приватный метод `simplify()`**
- Использует `std::gcd` (наибольший общий делитель) из заголовка `<numeric>` для сокращения дроби.
- Вызывается в конструкторе и после каждой арифметической операции.
- Следит за знаком: если знаменатель отрицательный, минус переносится в числитель.

**3) Конструктор**
- Принимает числитель и знаменатель.
- Если знаменатель равен 0, выбрасывает исключение `std::invalid_argument`.

**4) Оператор `*` (умножение)**
- Перегрузка умножения одной дроби на другую.
- Возвращает новый объект `Fraction`.

**5) Оператор `+` (сложение)**
- Перегрузка сложения двух дробей.
- Находит общий знаменатель, складывает и возвращает новый результат (автоматически сокращенный благодаря `simplify`).

**6) Оператор `<<` (вывод)**
- Дружественная функция для вывода в поток в формате `num/denom`.
