# Задание

Реализуйте класс User для системы регистрации, который автоматически присваивает уникальные ID каждому новому пользователю и подсчитывает их общее количество, используя статические члены.

**1) Приватные статические поля**
- static int nextId - хранит ID для следующего пользователя (изначально 1).
- static int usersCount - хранит текущее количество активных пользователей.

**2) Поля объекта**
- const int id - уникальный номер (задается в конструкторе и не меняется).
- std::string username.

**3) Конструктор**
- Принимает имя пользователя.
- Присваивает полю id значение nextId.
- Инкрементирует nextId.
- Инкрементирует usersCount.

**4) Деструктор**
- Декрементирует usersCount (уменьшает счетчик активных пользователей при удалении объекта).

**5) Статический метод getUsersCount()**
- Возвращает текущее количество пользователей.

**6) Методы доступа**
- getId() - возвращает ID.
- getName() - возвращает имя.

**7) методы изменения**
- setName(const std::string& newName) — устанавливает новое имя пользователя.
