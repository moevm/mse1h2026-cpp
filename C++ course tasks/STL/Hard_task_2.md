
# Задание

Вам нужно реализовать класс `Matrix`, который представляет матрицу целых чисел. Класс должен хранить данные в векторе векторов. Все методы должны использовать алгоритмы стандартной библиотеки (STL).

Класс должен содержать метожды:

**1) Конструктор по умолчанию `Matrix()`**
- **Формат:** `Matrix()`
- Создает пустую матрицу (без строк и столбцов)
- Поле `data` должно быть пустым вектором

**2) Конструктор с размерами `Matrix(size_t rows, size_t cols)`**
- **Формат:** `Matrix(3, 4)` - создает матрицу 3×4
- Принимает: количество строк `rows` и количество столбцов `cols`
- Создает матрицу указанного размера, заполненную нулями

**3) Конструктор от вектора векторов `Matrix(const std::vector<std::vector<int>>& input)`*
- **Формат:** `Matrix({{1, 2, 3}, {4, 5, 6}})` - создает матрицу 2×3
- Принимает: двумерный вектор для инициализации
- Инициализирует матрицу переданными данными

**4) Метод `fill(int value)`**
- **Формат:** `matrix.fill(5)` - заполняет все элементы значением 5
- Заполняет все элементы матрицы значением `value`
- Использовать: `std::fill` для каждой строки
- Ничего не возвращает

**5) Метод `sum() const`**
- **Формат:** `int total = matrix.sum()`
- Возвращает сумму всех элементов матрицы
- Использовать: `std::accumulate` для каждой строки и суммировать результаты
- Возвращает `int`
- Если матрица пуста, вернуть `0`

**6) Метод `minElement() const`**
- **Формат:** `int min = matrix.minElement()`
- Находит минимальный элемент во всей матрице
- Использовать: `std::min_element` для каждой строки и сравнивать результаты
- Возвращает `int`
- Если матрица пуста, вернуть `0`

**7) Метод `maxElement() const`**
- **Формат:** `int max = matrix.maxElement()`
- Находит максимальный элемент во всей матрице
- Использовать: `std::max_element` для каждой строки и сравнивать результаты
- Возвращает `int`
- Если матрица пуста, вернуть `0`

**8) Метод `countIf(std::function<bool(int)> predicate) const`**
- **Формат:** `int evenCount = matrix.countIf([](int x) { return x % 2 == 0; })`
- Подсчитывает количество элементов, удовлетворяющих предикату
- Использовать: `std::count_if` для каждой строки и суммировать результаты
- Возвращает `int`

**9) Метод `allOf(std::function<bool(int)> predicate) const`**
- **Формат:** `bool allPositive = matrix.allOf([](int x) { return x > 0; })`
- Проверяет, все ли элементы матрицы удовлетворяют предикату
- Использовать: `std::all_of` для каждой строки
- Возвращает `bool`
- Если матрица пуста, вернуть `true`

**10) Метод `anyOf(std::function<bool(int)> predicate) const`**
- **Формат:** `bool hasEven = matrix.anyOf([](int x) { return x % 2 == 0; })`
- Проверяет, существует ли хотя бы один элемент, удовлетворяющий предикату
- Использовать: `std::any_of` для каждой строки
- Возвращает `bool`
- Если матрица пуста, вернуть `false`

**11) Метод `sortRowsBySum()`**
- **Формат:** `matrix.sortRowsBySum()`
- Сортирует строки матрицы по возрастанию суммы элементов в строке
- Использовать: `std::sort` с компаратором, вычисляющим сумму через `std::accumulate`
- Ничего не возвращает

**12) Метод `print() const`**
- **Формат вывода:**
```cpp
    [    1    2    3 ]
    [    4    5    6 ]
    [    7    8    9 ]
    
```
- Каждая строка выводится в формате: `[` + элементы + `]`
- Каждый элемент занимает **4 позиции** (использовать `std::setw(4)`)
- Элементы разделяются пробелами
- После последнего элемента в строке пробел не ставится
- Ничего не возвращает
