
# Задание

Вам нужно реализовать класс `MessageQueue`, который представляет очередь сообщений с двусторонним доступом. Класс должен использовать `std::deque` для хранения строковых сообщений. Все методы должны использовать алгоритмы стандартной библиотеки (STL).

Класс должен содержать поле:
- `std::deque<std::string> messages` - двусторонняя очередь для хранения сообщений

Класс должен содержать методы:

**1) Конструктор по умолчанию `MessageQueue()`**
- **Формат:** `MessageQueue()`
- Создает пустую очередь сообщений
- Поле `messages` должно быть пустым `deque`

**2) Метод `sendFront(const std::string& message)`**
- **Формат:** `queue.sendFront("Hello")`
- Добавляет сообщение в начало очереди
- Использовать: `push_front`
- Ничего не возвращает

**3) Метод `sendBack(const std::string& message)`**
- **Формат:** `queue.sendBack("World")`
- Добавляет сообщение в конец очереди
- Использовать: `push_back`
- Ничего не возвращает

**4) Метод `receiveFront()`**
- **Формат:** `std::string msg = queue.receiveFront()`
- Удаляет и возвращает сообщение из начала очереди
- Если очередь пуста, возвращать пустую строку `""`
- Использовать: `front` и `pop_front`
- Возвращает `std::string`

**5) Метод `receiveBack()`**
- **Формат:** `std::string msg = queue.receiveBack()`
- Удаляет и возвращает сообщение из конца очереди
- Если очередь пуста, возвращать пустую строку `""`
- Использовать: `back` и `pop_back`
- Возвращает `std::string`

**6) Метод `peekFront() const`**
- **Формат:** `std::string msg = queue.peekFront()`
- Возвращает сообщение из начала очереди без удаления
- Если очередь пуста, возвращать пустую строку `""`
- Использовать: `front`
- Возвращает `std::string`

**7) Метод `peekBack() const`**
- **Формат:** `std::string msg = queue.peekBack()`
- Возвращает сообщение из конца очереди без удаления
- Если очередь пуста, возвращать пустую строку `""`
- Использовать: `back`
- Возвращает `std::string`

**8) Метод `isEmpty() const`**
- **Формат:** `bool empty = queue.isEmpty()`
- Проверяет, пуста ли очередь
- Использовать: `empty`
- Возвращает `bool`

**9) Метод `size() const`**
- **Формат:** `size_t count = queue.size()`
- Возвращает количество сообщений в очереди
- Использовать: `size`
- Возвращает `size_t`

**10) Метод `clear()`**
- **Формат:** `queue.clear()`
- Очищает очередь (удаляет все сообщения)
- Использовать: `clear`
- Ничего не возвращает

**11) Метод `receiveAll()`**
- **Формат:** `std::vector<std::string> all = queue.receiveAll()`
- Удаляет и возвращает все сообщения из очереди
- Возвращает `std::vector<std::string>` со всеми сообщениями
- Очередь после вызова должна стать пустой

**12) Метод `search(const std::string& substring) const`**
- **Формат:** `auto found = queue.search("Hello")`
- Ищет все сообщения, содержащие указанную подстроку
- Использовать: `std::copy_if` с `std::string::find`
- Возвращает `std::vector<std::string>` с найденными сообщениями

**13) Метод `filter(std::function<bool(const std::string&)> predicate) const`**
- **Формат:** `auto longMessages = queue.filter([](const std::string& s) { return s.length() > 5; })`
- Фильтрует сообщения по пользовательскому предикату
- Использовать: `std::copy_if`
- Возвращает `std::vector<std::string>` с отфильтрованными сообщениями

**14) Метод `printQueue() const`**
- **Формат вывода:**
```cpp
[1] Hello
[2] World
[3] Test
Total: 3 messages
```
- Выводит все сообщения в очереди от начала к концу
- Нумерация начинается с 1 (первый элемент - начало очереди)
- В конце выводит общее количество сообщений
- Ничего не возвращает
