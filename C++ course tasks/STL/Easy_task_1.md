
# Задание
Вам нужно написать программу, которая демонстрирует работу с вектором и использование основных алгоритмов стандартной библиотеки.

**1) Создайте вектор `numbers`**
- Создайте пустой вектор целых чисел `std::vector<int> numbers`
- С помощью цикла `for` добавьте в вектор числа от 1 до 10 включительно (используйте `push_back`)

**2) Сохраните исходный вектор в переменную `original`**
- Создайте копию вектора `numbers` в переменной `original`
- Это понадобится для дальнейших операций

**3) Найдите сумму всех элементов**
- Используйте алгоритм `std::accumulate` из библиотеки `<numeric>`
- Начальное значение суммы = 0
- Сохраните результат в переменную `sum` типа `int`

**4) Найдите среднее арифметическое**
- Разделите сумму на количество элементов
- Результат сохраните в переменную `average` типа `double`

**5) Найдите минимальный и максимальный элементы**
- Используйте алгоритмы `std::min_element` и `std::max_element`
- Они возвращают итераторы - разыменуйте их для получения значений
- Сохраните результаты в переменные `minValue` и `maxValue` типа `int`

**6) Подсчитайте количество четных чисел**
- Используйте алгоритм `std::count_if`
- **Создайте функцию `isEven`**, которая проверяет, является ли число четным
- Сохраните результат в переменную `evenCount` типа `int`

**7) Отсортируйте вектор `numbers` по убыванию**
- Используйте алгоритм `std::sort`
- **Создайте функцию `descending`**, которая сравнивает два числа так, чтобы сортировка была по убыванию

**8) Сохраните отсортированный вектор в переменную `sortedDesc`**
- Создайте копию отсортированного вектора `numbers` в переменной `sortedDesc`

**9) Удалите из вектора `numbers` все элементы меньше 5**
- Используйте алгоритм `std::remove_if` вместе с методом `erase`
- **Создайте функцию `lessThan5`**, которая возвращает `true`, если число меньше 5

**10) Сохраните итоговый вектор в переменную `filtered`**
- Создайте копию полученного вектора в переменной `filtered`

**11) Выведите все результаты**
- Выведите исходный вектор `original`
- Выведите `sum`, `average`, `minValue`, `maxValue`, `evenCount`
- Выведите отсортированный вектор `sortedDesc`
- Выведите отфильтрованный вектор `filtered`
