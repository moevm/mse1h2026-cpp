
# Задание

Дан класс `TaskQueue`, который управляет очередью задач. Нужно реализовать его методы для добавления, выполнения и управления задачами.

**1) Метод `addTask(const std::string& task)`**
- Принимает: строку с названием задачи
- Добавляет задачу в конец очереди `tasks` (используйте `push`)
- Ничего не возвращает

**2) Метод `addTasks(const std::vector<std::string>& newTasks)`**
- Принимает: вектор строк с новыми задачами
- Добавляет все задачи из вектора в конец очереди
- Использует метод `addTask` или прямой `push` в цикле
- Ничего не возвращает

**3) Метод `processNextTask()`**
- Принимает: ничего
- Проверяет, пуста ли очередь
- Если очередь пуста, возвращает строку "Нет задач для выполнения"
- Если очередь не пуста:
    - Получает следующую задачу (используйте `front`)
    - Удаляет её из очереди (используйте `pop`)
    - Добавляет задачу в вектор `completedTasks`
    - Возвращает строку с информацией о выполненной задаче в формате: `"Выполнена задача: <название>"`

**4) Метод `processTasks(int count)`**
- Принимает: целое число `count` - количество задач для выполнения
- Выполняет не более `count` задач (или сколько есть в очереди)
- Для каждой выполненной задачи:
    - Добавляет её в вектор `completedTasks`
    - Сохраняет результат в вектор строк
- Возвращает `std::vector<std::string>` - вектор с отчетами о выполненных задачах (в том же формате, что и `processNextTask`)

**5) Метод `peekNextTask() const`**
- Принимает: ничего
- Проверяет, пуста ли очередь
- Если очередь пуста, возвращает строку "Очередь пуста"
- Если очередь не пуста, возвращает следующую задачу **без удаления** из очереди (используйте `front`)

**6) Метод `isEmpty() const`**
- Принимает: ничего
- Возвращает `true`, если очередь пуста, иначе `false`

**7) Метод `getSize() const`**
- Принимает: ничего
- Возвращает текущий размер очереди (используйте `size`)

**8) Метод `clear()`**
- Принимает: ничего
- Очищает очередь задач (пока очередь не станет пустой, используя `pop`)
- Ничего не возвращает

**9) Метод `getCompletedTasks() const`**
- Принимает: ничего
- Возвращает вектор `completedTasks` -список всех выполненных задач


```cpp
class TaskQueue {
private:
    std::queue<std::string> tasks;
    std::vector<std::string> completedTasks;
    
public:
    TaskQueue() {}
    
    void addTask(const std::string& task);
    
    void addTasks(const std::vector<std::string>& newTasks);
    
    std::string processNextTask();
    
    std::vector<std::string> processTasks(int count);
    
    std::string peekNextTask() const;
    
    bool isEmpty() const;
    
    int getSize() const;
    
    void clear();

    std::vector<std::string> getCompletedTasks() const;
```
