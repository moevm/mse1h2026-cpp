
# Задание

Напишите программу, демонстрирующую базовую работу с `std::shared_ptr`.

**Вам необходимо:**
1. Создать `shared_ptr<int>` со значением, запрашиваемым с клавиатуры
2. Вывести значение
3. Создать две копии указателя, после каждой выводить значение
4. Вывести значения из всех трех указателей
5. Изменить значение через второй указатель и показать, что оно изменилось во всех, выведя значения из всех трех указателей
