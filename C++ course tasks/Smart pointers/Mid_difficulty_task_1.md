
# Задание

Вам необходимо реализовать класс `LinkedList`, представляющий односвязный список целых чисел. Для хранения узлов используется `std::unique_ptr`. Заготовка класса и структуры узлов даны, вы должны написать методы:

**1) Метод `push_front(int value)`**
- Принимает: целое число для добавления
- Создает новый узел с переданным значением
- Новый узел становится головой списка, его `next` указывает на старую голову
- Увеличивает счетчик `count` на 1
- Ничего не возвращает

**2) Метод `pop_front()`**
- Принимает: ничего
- Удаляет первый элемент списка
- Если список пуст, ничего не делает
- Уменьшает счетчик `count` на 1
- Ничего не возвращает

**3) Метод `insert(int value, size_t index)`**
- Принимает: значение для вставки и индекс позиции
- Вставляет новый элемент на указанную позицию
- Если `index` равен 0, работает как `push_front`
- Если `index` больше или равен текущему размеру, добавляет элемент в конец
- Увеличивает счетчик `count` на 1
- Ничего не возвращает

**4) Метод `erase(size_t index)`**
- Принимает: индекс удаляемого элемента
- Удаляет элемент по указанному индексу
- Если `index` вне диапазона (>= `size()`), ничего не делает
- Уменьшает счетчик `count` на 1
- Ничего не возвращает

**5) Метод `front()`**
- Принимает: ничего
- Возвращает ссылку на данные первого элемента (`head->data`)
- Если список пуст, выбрасывает исключение `std::out_of_range` с сообщением "List is empty"

**6) Метод `at(size_t index)`**
- Принимает: индекс элемента
- Возвращает ссылку на данные элемента по указанному индексу
- Если индекс вне диапазона, выбрасывает исключение `std::out_of_range` с сообщением "Index out of range"

**7) Метод `size() const`**
- Принимает: ничего
- Возвращает текущее количество элементов в списке (поле `count`)

**8) Метод `empty() const`**
- Принимает: ничего
- Возвращает `true`, если список пуст (`count == 0`), иначе `false`

**9) Метод `print() const`**
- Принимает: ничего
- Выводит все элементы списка в формате: `[1, 2, 3, 4]`
- Для пустого списка выводит: `[]`
- Ничего не возвращает

**Узел:**
```cpp
struct Node {
    int data;
    std::unique_ptr<Node> next;
    
    Node(int value) : data(value), next(nullptr) {}
};
```

**Класс:**
```cpp
class LinkedList {
private:
    std::unique_ptr<Node> head;
    size_t count;
    
public:
    LinkedList() : head(nullptr), count(0) {}
    
    void push_front(int value);
    
    void pop_front();
    
    void insert(int value, size_t index);
    
    void erase(size_t index);
    
    int& front();
    
    int& at(size_t index);
    
    size_t size() const;
    
    bool empty() const;
    
    void print() const;
};
```
