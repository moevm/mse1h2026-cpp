
# Задание

Вам необходимо реализовать класс `DoublyLinkedList`, представляющий двусвязный список целых чисел. Для хранения узлов используется `std::shared_ptr`. Заготовка класса и структуры узлов даны, вы должны написать методы:

**1) Метод `push_front(int value)`**
- Принимает: целое число для добавления
- Создает новый узел с переданным значением
- Новый узел становится головой списка
- Обновляет связи: новый узел указывает на старую голову как `next`, старая голова указывает на новый узел как `prev`
- Если список был пуст, новый узел становится и головой, и хвостом
- Увеличивает счетчик `count` на 1
- Ничего не возвращает

**2) Метод `push_back(int value)`**
- Принимает: целое число для добавления
- Создает новый узел с переданным значением
- Новый узел становится хвостом списка
- Обновляет связи: старый хвост указывает на новый узел как `next`, новый узел указывает на старый хвост как `prev`
- Если список был пуст, новый узел становится и головой, и хвостом
- Увеличивает счетчик `count` на 1
- Ничего не возвращает

**3) Метод `pop_front()`**
- Принимает: ничего
- Удаляет первый элемент списка
- Если список пуст, ничего не делает
- Перемещает `head` на следующий элемент
- Если новый `head` существует, обнуляет его `prev`
- Если список становится пустым, обнуляет `tail`
- Уменьшает счетчик `count` на 1
- Ничего не возвращает

**4) Метод `pop_back()`**
- Принимает: ничего
- Удаляет последний элемент списка
- Если список пуст, ничего не делает
- Перемещает `tail` на предыдущий элемент (через `lock()` от `prev`)
- Если новый `tail` существует, обнуляет его `next`
- Если список становится пустым, обнуляет `head`
- Уменьшает счетчик `count` на 1
- Ничего не возвращает

**5) Метод `insert(int value, size_t index)`**
- Принимает: значение для вставки и индекс позиции
- Вставляет новый элемент на указанную позицию
- Если `index == 0`, работает как `push_front`
- Если `index >= size()`, работает как `push_back`
- Иначе находит узел на позиции `index` и вставляет перед ним
- Обновляет соответствующие связи
- Увеличивает счетчик `count` на 1
- Ничего не возвращает

**6) Метод `at(size_t index)`**
- Принимает: индекс элемента
- Возвращает ссылку на данные элемента по указанному индексу
- Если индекс вне диапазона, выбрасывает исключение `std::out_of_range`

**7) Метод `size() const`**
- Возвращает текущее количество элементов в списке (поле `count`)

**8) Метод `get_node(size_t index) const`**
- Возвращает `std::shared_ptr<Node>` на узел по указанному индексу
- Если индекс вне диапазона, возвращает `nullptr`

**Узел:**
```cpp
struct Node {
    int data;
    std::shared_ptr<Node> next;
    std::weak_ptr<Node> prev; 
    Node(int value) : data(value), next(nullptr) {}
};
```

**Класс:**
```cpp
class DoublyLinkedList {
private:
    std::shared_ptr<Node> head;
    std::shared_ptr<Node> tail;
    size_t count;
    
public:
    DoublyLinkedList() : head(nullptr), tail(nullptr), count(0) {}
    
    void push_front(int value);
    
    void push_back(int value);
    
    void pop_front();
    
    void pop_back();
    
    void insert(int value, size_t index);
    
    int& at(size_t index);
    
    size_t size() const;
    
    std::shared_ptr<Node> get_node(size_t index) const;
};
```
