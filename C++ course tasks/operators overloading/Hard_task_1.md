# Задание

Вам нужно создать класс `Matrix3x3` для работы с матрицами 3x3 (вещественные числа). Класс должен хранить элементы матрицы в двумерном массиве `double m[3][3]`.

Реализовать класс необходимо полностью, включая конструкторы, перегрузку операторов, и вывод в заданном формате.

Конструкторы:
1. Конструктор по умолчанию (нулевая матрица)
2. Конструктор с 9 параметрами (заполнение по строкам):
```cpp
Matrix3x3(double a11, double a12, double a13,
double a21, double a22, double a23,
double a31, double a32, double a33);
```

Операторы:
1. Оператор () для доступа к элементам
	Принимает: индексы строки i и столбца j (целые числа от 0 до 2)  
	Возвращает: double
2. Арифметические операторы: +, -
	Принимают: константную ссылку на другую матрицу  
	Возвращают: новый объект Matrix3x3 - результат покомпонентного сложения или вычитания
3. Оператор * для умножения матриц
	Принимает: константную ссылку на другую матрицу  
	Возвращает: новый объект Matrix3x3 - результат матричного умножения
4. Оператор * для умножения матрицы на скаляр
	Принимает: вещественное число scalar  
	Возвращает: новый объект Matrix3x3 - исходная матрица, умноженная на скаляр
5. Оператор ~ для транспонирования
	Не принимает параметров  
	Возвращает: новый объект Matrix3x3 - транспонированную матрицу
6. Оператор || для вычисления определителя
	Не принимает параметров  
	Возвращает: вещественное число - определитель матрицы


Формат вывода:
- Каждая строка начинается с `[ ` и заканчивается ` ]`
- Элементы имеют фиксированный формат с **4 знаками после запятой**
- Элементы разделены запятой и пробелом: `, `
- После последнего элемента в строке запятая не ставится

Пример:
```cpp
[ 1.0000, 10.0000, 0.0000 ]  
[ 0.0000, 1.0000, 0.0000 ]  
[ 0.0000, 0.0000, 9999.9999 ]
```
