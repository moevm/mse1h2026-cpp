
## Задание

Дан класс `Point`, описывающий точку на двухмерной плоскости с координатами **`x`** и **`y`**. 
Вам необходимо дописать класс `Point`, добавив в него перегрузку двух операторов:

1. **Оператор `+`** должен складывать координаты двух точек и возвращать новую точку.
2. **Оператор `==`** должен сравнивать две точки на равенство (возвращать `true`, если координаты совпадают, иначе - false).
## Код:

```cpp
#include <iostream>

class Point {
private:
    double x, y;
    
public:
    Point(double x = 0, double y = 0) : x(x), y(y) {}
    
    void print() const
    {
        std::cout << "(" << x << ", " << y << ")";
    }
    
    // ваш код перегрузки операторов

};
