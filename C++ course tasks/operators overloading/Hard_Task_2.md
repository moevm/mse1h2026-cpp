# Задание

Вам нужно создать класс `RGB` для работы с цветом в формате RGB. Класс должен хранить три компоненты цвета (красный, зеленый, синий) в виде целых чисел от 0 до 255.

Реализовать класс необходимо полностью, включая конструкторы, перегрузку операторов, и вывод в заданном формате.

Конструкторы:
1. Конструктор по умолчанию - создает черный цвет (0, 0, 0)
2. Конструктор с параметрами - принимает три компоненты:
```cpp
RGB(int red, int green, int blue);
```
При создании значения должны приводиться к допустимому диапазону 0-255 (значения меньше 0 становятся 0, больше 255 становятся 255)
3. Конструктор копирования - копирование компонент

**Операторы:**
1. **Оператор [] для доступа к компонентам**
	Принимает: индекс компоненты (0 - красный, 1 - зеленый, 2 - синий)
	Возвращает: значение компоненты цвета
2. **Арифметические операторы: +, -**
	Принимают: константную ссылку на другой цвет
	Возвращает: новый объект `RGB` - результат покомпонентного сложения или вычитания
	требования: результат округлять вниз до целого, если он попадает в диапазон 0-255, до нуля если результат меньше нуля, до 255 если результат больше 255.
3. **Оператор * для смешивания цветов (компонентное умножение)**
	Принимает: константную ссылку на другой цвет
	Возвращает: новый объект `RGB` - результат покомпонентного умножения (каждая компонента = (c1 * c2) / 255)
	требования: результат округлять вниз до целого, если он попадает в диапазон 0-255, до нуля если результат меньше нуля, до 255 если результат больше 255.
4. **Оператор * для умножения цвета на яркость**
	Принимает: вещественное число brightness (от 0.0 до 1.0)
	Возвращает: новый объект `RGB` - исходный цвет, умноженный на яркость
	требования: результат округлять вниз до целого, если он попадает в диапазон 0-255, до нуля если результат меньше нуля, до 255 если результат больше 255.
5. **Оператор ! для получения негатива**
	Не принимает параметров (унарный оператор)
	Возвращает: новый объект `RGB` - негатив цвета (255 - R, 255 - G, 255 - B)
6. **Оператор ~ для перевода в grayscale**
	Не принимает параметров (унарный оператор)
	Возвращает: новый объект `RGB` - оттенки серого
	Формула: 0.299*R + 0.587*G + 0.114*B
	(значение округлить вниз до целого, если он попадает в диапазон 0-255, до нуля если результат меньше нуля, до 255 если результат больше 255, и присвоить всем трем компонентам)
7. **Операторы сравнения: == и !=**
	Принимает: константную ссылку на другой цвет
	Возвращает: `true` или `false`
	требования: цвета равны, если равны все три компоненты

**Формат вывода:**
- Выводятся три компоненты: красный, зеленый, синий
- Компоненты выводятся как целые числа (без дробной части)
- Компоненты разделены запятой и пробелом: `,`
- Вывод начинается с `[` и заканчивается `]`

**Пример:**
```cpp
[ 255, 0, 0 ]    // красный
[ 0, 255, 0 ]    // зеленый
[ 0, 0, 255 ]    // синий
```
