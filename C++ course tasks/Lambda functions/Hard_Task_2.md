
# Задание

Вам нужно реализовать класс `TextAnalyzer`, который позволяет анализировать текст: подсчитывать слова и символы по различным критериям. Класс хранит текст в виде строки и предоставляет методы для его анализа с использованием лямбда-выражений.

Методы должны выглядеть так:

**1) Конструктор `TextAnalyzer(const std::string& inputText)`**
- Принимает: строку с текстом для анализа
- Сохраняет текст в поле `text`
- Вызывает приватный метод `splitIntoWords()` для разбиения текста на слова
- Ничего не возвращает (конструктор)

**2) Приватный метод `splitIntoWords()`**
- Принимает: ничего
- Разбивает сохраненный текст на отдельные слова (используя `std::istringstream`)
- Очищает слова от знаков препинания (запятые, точки, восклицательные знаки и т.д.)
- Заполняет вектор `words` полученными словами
- Ничего не возвращает (`void`)

**3) Метод `countWords(bool (*predicate)(const std::string&)) const`**
- Принимает: указатель на функцию-предикат, которая принимает строку и возвращает `bool`
- **Создает лямбда-выражение**, которое будет применяться к каждому слову для подсчета
- В цикле проходит по всем словам, применяет переданный предикат
- Подсчитывает количество слов, для которых предикат вернул `true`
- Возвращает `int` - количество слов, удовлетворяющих условию

**4) Метод `countChars(bool (*predicate)(char)) const`**
- Принимает: указатель на функцию-предикат, которая принимает символ и возвращает `bool`
- **Создает лямбда-выражение** для подсчета символов, удовлетворяющих условию
- В цикле проходит по всем символам текста, применяет предикат к каждому символу
- Возвращает `int` - количество символов, удовлетворяющих условию

**5) Метод `printStats() const`**
- Принимает: ничего
- Выводит в консоль базовую статистику по тексту в формате:
```
Text Statistics:
Total words: 15
Total characters: 87
```
- Ничего не возвращает (`void`)


```cpp
class TextAnalyzer {
private:
    std::string text;
    std::vector<std::string> words;
    
    void splitIntoWords();
    
public:
    TextAnalyzer(const std::string& inputText);
    
    int countWords(bool (*predicate)(const std::string&)) const;
    
    int countChars(bool (*predicate)(char)) const;
    
    void printStats() const;
};
```
