
# Задание

Вам нужно написать программу, реализовав четыре различных типа лямбда-выражений:

Лямбда без параметров - должна выводить строку "Hello, Lambda!" и вызываться сразу после создания.
- Лямбда не должна принимать параметры
- Должна быть вызвана сразу после создания (без сохранения в переменную)

Лямбда с параметрами - должна принимать два целых числа и возвращать их сумму.
- Лямбда должна принимать два целых числа
- Должна возвращать их сумму
- Сохранить лямбду в переменную `sum` для последующего вызова

Лямбда с захватом переменной по значению - должна захватывать переменную `multiplier` и умножать переданное число на нее.
- Захватить переменную `multiplier` по значению
- Принимать один целочисленный параметр
- Возвращать произведение параметра и захваченного `multiplier`

Лямбда с захватом переменной по ссылке - должна захватывать переменную `counter` по ссылке и увеличивать ее на 1 при каждом вызове.
- Захватить переменную `counter` по ссылке
- Не принимать параметров
- Увеличивать `counter` на 1 при каждом вызове
