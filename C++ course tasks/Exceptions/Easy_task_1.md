
# Задание

Вам нужно реализовать класс `Calculator`, который выполняет арифметические операции и выбрасывает исключения при некорректных операциях. Все методы должны обрабатывать исключительные ситуации с использованием механизма исключений C++.

Класс калькулятор должен содержать следующие методы:

**1) Конструктор по умолчанию `Calculator()`**
- **Формат:** `Calculator calc;`
- Создает объект калькулятора
- Ничего не возвращает

**2) Метод `divide(double a, double b) const`**
- **Формат:** `double result = calc.divide(10, 2);`
- Делит первое число на второе
- **Если `b` равно `0.0`, метод должен выбрасывать исключение**
- Использовать: `throw std::runtime_error("Деление на ноль невозможно");`
- Возвращает `double` - результат деления

**3) Метод `calculate(const std::string& operation, double a, double b) const`**
- **Формат:** `double result = calc.calculate("+", 5, 3);`
- Принимает строку с операцией (`+`, `-`, `*`, `/`) и два числа
- Для операции `/` вызывает метод `divide`
- Для операций `+`, `-`, `*` выполняет соответствующее действие
- **Если операция неизвестна, выбрасывает исключение `std::invalid_argument`**
- Возвращает `double` - результат операции
