
# Задание

Вам нужно реализовать класс `RGB`, который представляет цвет в формате RGB (Red, Green, Blue). Каждая компонента цвета хранится как целое число в диапазоне от 0 до 255.

Реализуйте класс с полями для хранения трех компонент цвета. Класс должен иметь приведённые ниже методы. Все проверки на корректность операций должны выполняться внутри методов, и при обнаружении ошибок должны выбрасываться исключения.

**Методы для реализации:**

**1) Конструкторы**
- `RGB()` - конструктор по умолчанию (создает черный цвет 0, 0, 0)
- `RGB(int red, int green, int blue)` - создает цвет из трех компонент

**2) Доступ к компонентам**
- `int getRed() const` - возвращает красную компоненту
- `int getGreen() const` - возвращает зеленую компоненту
- `int getBlue() const` - возвращает синюю компоненту
- `void setRed(int value)` - устанавливает красную компоненту
- `void setGreen(int value)` - устанавливает зеленую компоненту
- `void setBlue(int value)` - устанавливает синюю компоненту
- `int get(size_t index) const` - возвращает компоненту по индексу (0 - red, 1 - green, 2 - blue)
- `void set(size_t index, int value)` - устанавливает компоненту по индексу

**3) Арифметические операции**
- `RGB add(const RGB& other) const` - смешивание цветов (покомпонентное сложение с ограничением 0-255)
- `RGB subtract(const RGB& other) const` - вычитание цветов (покомпонентное вычитание с ограничением 0-255)
- `RGB multiply(double factor) const` - умножение яркости (каждая компонента умножается на factor, результат 0-255)

Определите самостоятельно, в каких ситуациях нужно выбрасывать исключения и какие типы исключений использовать.

**Ситуации, требующие выброса исключений:**

**1) В конструкторе от трех чисел**
- Если значение `red` меньше 0 или больше 255 - выбросить `std::out_of_range` с сообщением "Красная компонента должна быть в диапазоне 0-255"
- Если значение `green` меньше 0 или больше 255 - выбросить `std::out_of_range` с сообщением "Зеленая компонента должна быть в диапазоне 0-255"
- Если значение `blue` меньше 0 или больше 255 - выбросить `std::out_of_range` с сообщением "Синяя компонента должна быть в диапазоне 0-255"

**2) В методах установки компонент**
- В `setRed`: если значение меньше 0 или больше 255 - выбросить `std::out_of_range` с сообщением "Красная компонента должна быть в диапазоне 0-255"
- В `setGreen`: если значение меньше 0 или больше 255 - выбросить `std::out_of_range` с сообщением "Зеленая компонента должна быть в диапазоне 0-255"
- В `setBlue`: если значение меньше 0 или больше 255 - выбросить `std::out_of_range` с сообщением "Синяя компонента должна быть в диапазоне 0-255"

**3) В методах доступа по индексу**
- В `get`: если индекс не равен 0, 1 или 2 - выбросить `std::out_of_range` с сообщением "Индекс должен быть 0 (red), 1 (green) или 2 (blue)"
- В `set`: если индекс не равен 0, 1 или 2 - выбросить `std::out_of_range` с сообщением "Индекс должен быть 0 (red), 1 (green) или 2 (blue)"
- В `set`: если значение меньше 0 или больше 255 - выбросить `std::out_of_range` с сообщением "Значение компоненты должно быть в диапазоне 0-255"

**4) В методе `multiply`**
- Если коэффициент `factor` меньше 0 - выбросить `std::invalid_argument` с сообщением "Коэффициент умножения не может быть отрицательным"
