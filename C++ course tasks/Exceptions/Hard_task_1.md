
# Задание

Вам нужно реализовать класс `Matrix3x3`, который представляет матрицу целых чисел соответствующего размера. 

Реализуйте класс `std::vector<std::vector<int>> data` для хранения матрицы. Класс должен иметь приведённые ниже методы. Все проверки на корректность операций должны выполняться внутри методов, и при обнаружении ошибок должны выбрасываться исключения.

**Методы для реализации:**

**1) Конструкторы**
- `Matrix3x3()` - конструктор по умолчанию (создает единичную матрицу - на главной диагонали 1, остальные 0)
- `Matrix3x3(const std::vector<std::vector<int>>& input)` - создает матрицу из вектора векторов 3×3

**2) Доступ к элементам**
- `int get(int i, int j) const` - возвращает элемент матрицы по индексам
- `void set(int i, int j, int value)` - устанавливает значение элемента по индексам

**3) Арифметические операции**
- `Matrix3x3 add(const Matrix3x3& other) const` - сложение матриц (поэлементное)
- `Matrix3x3 subtract(const Matrix3x3& other) const` - вычитание матриц (поэлементное)

**4) Операции для работы с матрицей**
- `Matrix3x3 transpose() const` - возвращает транспонированную матрицу
- `int trace() const` - вычисляет след матрицы (сумму элементов на главной диагонали)

**Ситуации, требующие выброса исключений:**

**1) В конструкторе от вектора векторов**
- Если входной вектор пуст - выбросить `std::invalid_argument` с сообщением "Входной вектор не может быть пустым"
- Если количество строк не равно 3 - выбросить `std::invalid_argument` с сообщением "Матрица должна содержать ровно 3 строки"
- Если любая из строк имеет размер не 3 - выбросить `std::invalid_argument` с сообщением "Каждая строка должна содержать ровно 3 элемента"
- Если строки имеют разную длину - выбросить `std::invalid_argument` с сообщением "Все строки должны быть одинаковой длины"

**2) В методах доступа `get` и `set`**
- Если индекс строки `i` меньше 0 или больше 2 - выбросить `std::out_of_range` с сообщением "Индекс строки должен быть от 0 до 2"
- Если индекс столбца `j` меньше 0 или больше 2 - выбросить `std::out_of_range` с сообщением "Индекс столбца должен быть от 0 до 2"
